## Uso académico y continuidad del proyecto

Este repositorio reúne los artefactos técnicos desarrollados y modificados durante el proyecto de título asociado a OpenGlove. Su propósito es facilitar la revisión académica, la reproducción de la implementación y la continuidad del trabajo por futuros desarrolladores o memoristas.

El código, diagramas y documentación se distribuyen bajo la licencia MIT indicada en el archivo LICENSE.
